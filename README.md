# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**.
This project showcases my skills, projects, and contact information in a clean and user-friendly design.

---

## 🚀 Live Demo

🔗 https://mr-vijay16.github.io/Portfolio-Vijay/

---

## 📌 Features

* 📱 Fully Responsive Design (Mobile + Desktop)
* 🎯 Smooth Scrolling Navigation
* 🍔 Mobile Hamburger Menu with Overlay
* ✖ Toggle Menu (Hamburger → Close icon)
* 🧭 Active Section Highlight on Scroll
* 💼 Projects Showcase Section
* 🧑‍💻 About Me Section
* 📬 Contact Form UI
* 📄 Resume Download/View Button
* ✨ Clean UI with modern styling

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Media Queries)
* JavaScript (DOM Manipulation)

## ⚙️ How It Works

* **Navigation Menu**

  * Toggles on mobile using JavaScript
  * Overlay added for better UX
  * Auto closes on link click

* **Scroll Behavior**

  * Highlights active section
  * Smooth scroll to sections

* **Responsive Design**

  * Media queries used for mobile layout
  * Optimized spacing and alignment
